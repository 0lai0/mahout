## Problem

No upfront memory check exists. Users hit OOM mid-run after long waits (related to #1262).

## Scope

Pure Rust function, no side effects:

```rust
pub fn estimate_memory(
    encoding: Encoding,
    num_qubits: u32,
    batch_size: usize,
    dtype: Precision,
    prefetch_depth: usize,
) -> MemoryEstimate
```

Memory model:

- **CPU prefetch pool:** `prefetch_depth × batch_size × sample_size × bytes_per_elem`
- **GPU state buffer:** `2 × batch_size × state_len × complex_bytes` (adjust per encoding if needed)
- Optional term for Parquet streaming chunk buffer
